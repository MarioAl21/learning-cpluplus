Plantillas

Otro recurso que ofrece c++ para ganar generalidad en 
la definición de soluciones.
Permiten declara funciones dejando sin especificar el tipo 
de sus parámetros o, en el caso de las clases, sus atributos.

Plantillas de funciones
Modelo de una función que el compilador c++ usará para 
construir diferentes versiones de una misma función.

Sintaxis: template <class T1, class T2, ..., class T3>
